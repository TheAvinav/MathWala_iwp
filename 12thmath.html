Relations And Functions

  1. Relation (R): R ⊆ A × B
  2. Domain (D): D = { x ∣ (x, y) ∈ R}
  3. Codomain (C): C = B
  4. Range (R): R = { y ∣ ∃ x ∈ D, (x, y) ∈ R}
  5. Cartesian Product (A × B): A × B = {(a, b) ∣ a ∈ A and b ∈ B}
  6. Vertical Line Test: The vertical line test confirms that a relation is a function if each vertical line crosses the graph at most once.
  7. One-to-One Function: f(x1​) = f(x2​) ⇒ x1​ = x2​ only
  8. Onto Function (Surjective): Every element in the codomain has at least one preimage.
  9. Bijective: Both one-to-one and onto.
  10. Composition of Functions (f o g): (fog)(x) = f( g(x))

Trigonometry

  • Tangent & Cotangent Identities
    • tan(θ) = sin(θ)/cos(θ)
    • cot(θ) = cos(θ)/sin(θ)
  • Reciprocal Identities
    • csc(θ) = 1/sin(θ)
    • sec(θ) = 1/cos(θ)
    • cot(θ) = 1/tan(θ)
  • Pythagorean Identities
    • sin²(θ) + cos²(θ) = 1
    • tan²(θ) + 1 = sec²(θ)
    • 1 + cot²(θ) = csc²(θ)
  • Sum and Difference Formulas
    • sin(α ± β) = sin(α)cos(β) ± cos(α)sin(β)
    • cos(α ± β) = cos(α)cos(β) ∓ sin(α)sin(β)
  • Double Angle Formulas
    • sin(2θ) = 2sin(θ)cos(θ)
    • cos(2θ) = cos²(θ) - sin²(θ)

Matrices

  • Addition and Subtraction of Matrices
    • A + B = B + A
    • (A + B) + C = A + (B + C)
    • k(A + B) = kA + kB
  • Multiplication of Matrices
    • Generally, AB ≠ BA (Not commutative)
    • (AB)C = A(BC) (Associative Law)
    • Distributive Law: A(B + C) = AB + AC
    • Multiplicative Identity: AI = IA = A
  • Properties of Transpose of Matrix
    • (Aᵀ)ᵀ = A
    • (kA)ᵀ = kAᵀ
    • (A + B)ᵀ = Aᵀ + Bᵀ
    • (AB)ᵀ = BᵀAᵀ
  • Symmetric & Skew-Symmetric Matrices
    • Symmetric Matrix: If Aᵀ = A
    • Skew-Symmetric Matrix: If Aᵀ = -A
    • For any square matrix A, (A + Aᵀ) is symmetric.
  • Inverse of a Matrix & its Properties
    • If AB = BA = I, then B = A⁻¹
    • (A⁻¹)⁻¹ = A
    • (AB)⁻¹ = B⁻¹A⁻¹
    • Important: (A + B)⁻¹ ≠ A⁻¹ + B⁻¹
  • Properties of Determinants
    • det(I) = 1
    • |Aᵀ| = |A|
    • |AB| = |A| |B|
    • |A⁻¹| = 1 / |A|
    • |kA| = kⁿ|A|, where n is the order of the matrix.

Differentiation

  • Exponential and Logarithmic Formulas
    • d/dx (aˣ) = aˣ ln(a)
    • d/dx (eˣ) = eˣ
    • d/dx (logₐ(x)) = 1 / (x ln(a))
    • d/dx (ln(x)) = 1/x
  • Trigonometric Formulas
    • d/dx (sin(x)) = cos(x)
    • d/dx (cos(x)) = -sin(x)
    • d/dx (tan(x)) = sec²(x)
    • d/dx (cot(x)) = -csc²(x)
    • d/dx (sec(x)) = sec(x)tan(x)
    • d/dx (csc(x)) = -csc(x)cot(x)
  • Inverse Trigonometric Formulas
    • d/dx (sin⁻¹(x)) = 1 / √(1 - x²)
    • d/dx (cos⁻¹(x)) = -1 / √(1 - x²)
    • d/dx (tan⁻¹(x)) = 1 / (1 + x²)
    • d/dx (cot⁻¹(x)) = -1 / (1 + x²)
    • d/dx (sec⁻¹(x)) = 1 / (|x|√(x² - 1))
    • d/dx (csc⁻¹(x)) = -1 / (|x|√(x² - 1))
  • Hyperbolic Formulas
    • d/dx (sinh(x)) = cosh(x)
    • d/dx (cosh(x)) = sinh(x)
    • d/dx (tanh(x)) = sech²(x)
  • Chain Rule
    • dy/dx = (dy/du) × (du/dx)

Integration

  • Basic Standard Integrals
    • ∫ xⁿ dx = (xⁿ⁺¹ / (n+1)) + C
    • ∫ (1/x) dx = ln|x| + C
    • ∫ eˣ dx = eˣ + C
    • ∫ sin(x) dx = -cos(x) + C
    • ∫ cos(x) dx = sin(x) + C
    • ∫ sec²(x) dx = tan(x) + C
  • Integrals for Inverse Trigonometric Functions
    • ∫ dx / √(a² - x²) = sin⁻¹(x/a) + C
    • ∫ dx / (a² + x²) = (1/a)tan⁻¹(x/a) + C
    • ∫ dx / (x√(x² - a²)) = (1/a)sec⁻¹(x/a) + C
  • Special Integrals (Advanced Forms)
    • ∫ dx / (x² - a²) = (1/2a) ln|(x-a)/(x+a)| + C
    • ∫ dx / √(x² + a²) = ln|x + √(x² + a²)| + C
    • ∫ √(a² - x²) dx = (x/2)√(a² - x²) + (a²/2) sin⁻¹(x/a) + C
  • Integration by Parts
    • Rule: ∫ u v dx = u ∫ v dx - ∫ [ (du/dx) ∫ v dx ] dx
    • Tip: Use the ILATE rule to choose the function 'u'.
  • Properties of Definite Integrals
    • ∫[a to b] f(x) dx = F(b) - F(a)
    • ∫[0 to a] f(x) dx = ∫[0 to a] f(a-x) dx (King's Property)
    • If f(x) is an odd function, then ∫[-a to a] f(x) dx = 0

Vector Algebra

  • Basic Concepts
    • Magnitude: For a⃗ = xî + yĵ + zk̂, |a⃗| = √(x² + y² + z²)
    • Unit Vector: â = a⃗ / |a⃗|
    • Direction Cosines: l = cos(α), m = cos(β), n = cos(γ)
    • Property: l² + m² + n² = 1
  • Dot Product (Scalar Product)
    • Definition: a⃗ · b⃗ = |a⃗| |b⃗| cos(θ)
    • For Perpendicular Vectors: If a⃗ ⊥ b⃗, then a⃗ · b⃗ = 0
    • Angle Between Vectors: cos(θ) = (a⃗ · b⃗) / (|a⃗| |b⃗|)
    • Projection of a⃗ on b⃗: (a⃗ · b⃗) / |b⃗|
  • Cross Product (Vector Product)
    • Definition: a⃗ × b⃗ = (|a⃗| |b⃗| sin(θ)) n̂
    • For Parallel Vectors: If a⃗ ∥ b⃗, then a⃗ × b⃗ = 0
    • Area of Parallelogram: |a⃗ × b⃗|
    • Area of Triangle: ½ |a⃗ × b⃗|
  • Scalar Triple Product
    • Definition: [a⃗ b⃗ c⃗] = (a⃗ × b⃗) · c⃗
    • Geometric Meaning: Represents the volume of the parallelepiped.
    • Condition for Coplanarity: Three vectors are coplanar if [a⃗ b⃗ c⃗] = 0.

3D Geometry

  • The Circle
    • Standard Equation: (x - h)² + (y - k)² = r²
    • Center: (h, k)
    • Radius: r
    • General Equation: x² + y² + 2gx + 2fy + c = 0
    • Center from General: (-g, -f)
    • Radius from General: √(g² + f² - c)

A Circle

  • The Parabola
    • Standard Equation: y² = 4ax
    • Vertex: (0, 0)
    • Focus: (a, 0)
    • Directrix: x = -a
    • Length of Latus Rectum: 4a

A Parabola

  • The Ellipse
    • Standard Equation: x²/a² + y²/b² = 1
    • Foci: (±c, 0), where c² = a² - b²
    • Eccentricity: e = c/a (e < 1)
    • Length of Latus Rectum: 2b²/a

Permutations and Combinations

  • Permutations (Arrangements)
    • Arranging r from n: nPr = n! / (n-r)!
    • With identical items: n! / (n₁! n₂!)
    • In a circle: (n-1)!
  • Combinations (Selections)
    • Selecting r from n: nCr = n! / (r!(n-r)!)
    • Key Property: nCr = nC(n-r)
    • Pascal's Rule: nCr + nC(r-1) = (n+1)Cr
  • Binomial Distribution
    • Probability: P(X=r) = nCr · pʳ · qⁿ⁻ʳ
    • Mean (μ) = np
    • Variance (σ²) = npq
  • Basic Probability
    • Probability of Event E: P(E) = n(E) / n(S)
    • Complement: P(E') = 1 - P(E)
    • Addition Rule: P(A U B) = P(A) + P(B) - P(A ∩ B)
  • Conditional Probability
    • Definition: P(A|B) = P(A ∩ B) / P(B)
    • Multiplication Rule: P(A ∩ B) = P(B) · P(A|B)
    • For Independent Events: P(A ∩ B) = P(A) · P(B)
  • Bayes' Theorem
    • P(Ei|A) = [P(Ei) · P(A|Ei)] / P(A)
